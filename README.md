# ASCII ART 
简单ASCII art显示。

## 截图
![screen-1](https://github.com/haifenghuang/ascii_art/blob/master/screenshot.png)

## 特性
1. 简单易用
2. 支持自定义ASCII ART
3. 系统自带5种ASCII ART
4. 使用类似类的方式访问其API
5. 支持两种不同的字体间的互相MAPPING
6. 支持颜色

## 缺点
1. 不支持多字节
2. 所有的字体必须等长等宽

## 运行环境
  在MacOSX环境下，xcode运行通过

## 使用
API使用请参照`main.c`
